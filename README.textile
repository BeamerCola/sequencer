h1. Digital MIDI Step Sequencer

!http://beamercola.github.io/sequencer/images/seq_lg.png!

An Arduino-based digital MIDI step sequencer.

* The first row is your standard note assignment
* The second row is velocity (turned off in this video)
* The third row is measure based global transposition
