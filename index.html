An Open Source Arduino Sequencer

  • MIDI output
  • Tempo control
  • Variable 8 step
  • Velocity sliders
  • Multiplexer city
  • Compatible with all Arduinos

Measure Based Note Transposition

What makes this sequencer special to me is the third row. It steps once the last note is triggered on the above rows. It solely transposes the note assignment row for each measure. Allowing you to create extremely dynamic sequences.
